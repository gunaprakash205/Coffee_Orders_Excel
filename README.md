Coffee Orders Excel Project.


Overview

This project is an Excel-based coffee orders analysis project.

The dataset contains customer orders, product details, and sales information used for data analysis and reporting in Microsoft Excel.


File Included.

coffeeOrdersProject.xlsx — Excel workbook containing the dataset and analysis.


Features.

Data cleaning in Excel,
Pivot Tables,
Charts and graphs,
Sales analysis,
Customer order tracking.


Skills Used.

Microsoft Excel,
Pivot Tables,
Excel formulas,
Data visualization.


Insights.

Best-selling coffee products,
Revenue analysis,
Customer purchase trends,
Order summaries.


Author.

Created as a practice project for learning Excel and data analysis.
